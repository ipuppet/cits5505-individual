Here are some of the projects I have worked on:
#1. ipuppet/data-filter-project

This is an application that helps users filter data from Excel files by creating custom rules. It allows users to upload Excel files and apply filters to the data based on various criteria. It provides a user interface for selecting the data to be filtered and the criteria for filtering. The application supports various filtering options, including AND/OR conditions, field matching, and regular expressions. It is built using Python and Django, and it uses Pandas for data manipulation.

#2. ipuppet/iot_server

This is the backend of Home Automation Platform which allows users to control their smart home devices and create automation rules. It provides a RESTful API and WebSocket support for real-time communication. It is built using Python and Django, and it uses SQLite as the database.

#3. ipuppet/clipsync

This is a cross-platform clipboard synchronization tool that allows users to sync their clipboard across multiple devices. It is built using Go, and it use AES encryption to ensure the security of the clipboard data. The application is designed to be lightweight and easy to use.

#4. ipuppet/CAIO

This is an application that allows users to create and manage their clipboard. It provides a user-friendly interface for copying, pasting, and organizing clipboard items. The application is built based on JSBox, a JavaScript runtime environment for iOS.
